정규표현식 나타내기

/ 정규표현식 패턴입력 위치 / 플래그 위치 (gimsuy)

플래그 종류

  1. global : 일치하는 모든 것 찾기 (g 없으면 첫 번째 결과만 찾음)
  2. case insensitive : 대소문자 구분 없이 검색
  3. multiline : 다중 행 모드 (^과 $ 작동 방식에만 영향 줌)
  4. single line : .이 개행 문자 \n도 포함하도록 모드 활성화
  5. unicode : 유니코드 전체 지원
  6. sticky : 특정 위치에서 검색하는 모드

Groups and ranges (그룹과 범위)

  1. | : 또는
  2. ( ) : 그룹
  3. [ ] : 문자셋, 괄호안의 어떤 문자든지
  4. [^] : 부정 문자셋, 광호안의 어떤 문자가 아닐 때
  5. (?:) : 해당 문자를 찾기만 하고 그룹으로 기억하지 않음
예제1 : (?:a|b|c) 와 [abc] 는 같음. 한 글자씩 찾기
예제2 : [abcd]와 [a-d]는 같음. 한 글자씩
예제3 : [a-zA-Z0-9] a부터z A부터Z 0부터9까지 한 글자씩 다 찾기
예제4 : [^a-zA-Z0-9] a부터z A부터Z 0부터9까지 제외하고 한 글자씩 다 찾기

Quantifiers (수량자) (찾을 문자 뒤에 표기)

  1. ? : 없거나 하나 있거나
  2. * : 없거나 하나 이상이거나
  3. + : 하나 또는 그 이상
  4. {n} : n번 반복
  5. {min, } : 최소
  6. {min, max} : 최소 ~ 최대
예제1 : gra?y a가 있거나 없는 경우 gry gray 찾음
예제2 : gra*y gry gray graaay 등 다 찾음
예제3 : gra+y gray graaay 등 다 찾음
예제4 : gra{2}y graay 찾음
예제5 : gra{1,}y gray graay graaay 등 다 찾음
예제6 : gra{2.3}y graay graaay 찾음

Boundary-type (경계 타입)

  1. \b : 단어 경계 (문자 앞뒤)
  2. \B : 단어 경계 아님 (문자 앞뒤)
  3. ^ : 문장 시작 (문자 앞) (m플래그 영향 받음)
  4. $ : 문장 끝 (문자 뒤) (m플래그 영향 받음)
예제1 : Ya : Ya 모두 찾음
예제2 : \bYa : 단어 중 맨앞에서 쓰인 Ya만 찾음 Ya, YaYaYa중 맨 앞
예제3 : Ya\b : 단어 중 맨뒤에서 쓰인 Ya만 찾음 Ya, YaYaYa중 맨 뒤
예제4 : Ya\B : 단어 중 맨뒤에서 쓰인 Ya를 제외하고 찾음 YaYaYa중 앞에 두 개
예제5 : ^Ya : m 플래그 모드에서 문장 한 줄에서 맨 앞에 쓰인 Ya만 찾음 (m모드 아니면 전체 행에서 맨 앞)
예제6 : Ya$ : m 플래그 모드에서 문장 한 줄에서 맨 뒤에 쓰인 Ya만 찾음 (m모드 아니면 전체 행에서 맨 뒤)

Character classes (캐릭터 클래스)

  1. \특문 : 특수문자를 찾기
  2. . : 어떤 모든 글자 (s모드인 경우 줄바꿈 문자도 포함)
  3. \d : digit 숫자 찾기
  4. \D : digit 숫자 아님 찾기
  5. \w : word 문자 찾기
  6. \W : word 문자 아님 찾기
  7. \s : space 공백 찾기
  8. \S : space 공백 아님 찾기
예제1 : \. : .을 찾고 싶을 때
예제2 : \? : ?를 찾고 싶을 때
예제3 : \[\]\{\} : []{} 찾을 때
예제4 : \d : 모든 숫자 찾기
예제5 : \D : 숫자 제외 모든 문자 찾기

전후방탐색

< 후방탐색 : 뒤에서 앞으로 탐색
= 긍정형 : 포함된 걸 탐색
! 부정형 : 포함되지 않은 걸 탐색
  1. ?= : 긍정형 전방탐색
  2. ?! : 부정형 전방탐색
  3. ?<= : 긍정형 후방탐색
  4. ?<! : 부정형 후방탐색
예제1 : (?=.*[a-z]) : 앞에서부터 탐색하여 모든 문자0개 이상, a부터 z까지 소문자 알파벳이 들어있으면 탐색하여 그룹화
예제2 : 8~15자 숫자,특문,대문자 하나 이상 포함 비밀번호
^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{8,15}$

Q1 전화번호 다 찾기

answer : /\d{2,3}[- .]\d{3}[ -.]\d{3,4}/gm

Q2 이메일 다 찾기

answer : /[a-zA-Z0-9._+-]+@[a-zA-Z0-9._+-]+\.[a-zA-Z0-9.]+/gm

Q3 URL 유튜브 주소 다 찾기

answer : /(https?:\/\/)?(www\.)?youtube.be\/([a-zA-Z0-9_-]{11})/gm

Q3 URL 유튜브 주소에서 뒤에 id값만 찾기

answer : /(?:https?:\/\/)?(?:www\.)?youtube.be\/([a-zA-Z0-9_-]{11})/gm
